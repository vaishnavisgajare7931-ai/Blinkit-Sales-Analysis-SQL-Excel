Project Overview :

The Blinkit Sales Analysis project is designed to simulate a real world retail and quick commerce sales scenario. The objective of this project is to analyze raw sales data and convert it into meaningful insights that support business decision making.

This project demonstrates the complete data analysis workflow using SQL and Excel starting from raw data to final insights.

Tools and Technologies Used :

SQL for data querying ,analysis and aggregations Microsoft Excel for data cleaning pivot tables charts and dashboards GitHub for project documentation and version control

Dataset Description :

The dataset is used only for learning and analytical practice.

Key Analysis Performed :

Data cleaning and validation using Excel Sales trend analysis using SQL queries Region wise revenue and quantity analysis Identification of top performing and low performing regions Monthly sales trend analysis Customer demand behavior analysis

KPIs and Insights :

Total Sales and Total Quantity Sold Month over Month Sales Growth Top Regions by Revenue Regional contribution to total sales High demand periods in Blinkit sales

These insights help understand sales performance demand patterns and regional strengths.

Excel Dashboard Overview :

Pivot tables for structured analysis Charts for sales and trend visualization Region wise performance comparison Business friendly dashboard for quick insights

Business Recommendations :

Focus marketing on high performing regions Improve supply planning during peak demand periods Optimize inventory for low performing regions Plan promotions based on sales trends
